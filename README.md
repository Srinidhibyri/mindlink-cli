# MindLink (CLI Version)

> A simple system I built to manage tasks, thoughts, and daily execution in one place.

---

## Problem

I noticed a pattern in my daily routine:

- I forget important tasks  
- I delay things even when I know they matter  
- Too many small things stay in my head  
- By the end of the day, it feels unstructured  

So I decided to try solving it by building something small.

---

## What is MindLink?

MindLink is a small project I built to experiment with this idea:

→ Can I create a simple system to capture and organize my tasks and thoughts?

Right now, it works as a basic command-line tool that helps me:

At its core, I’m trying to explore a simple “Thought → Action” system.

- Store tasks  
- Track deadlines  
- Keep small memory notes  
- Plan what I want to do today  

---

## Current Version (CLI)

This is the Command Line Interface (CLI) version.

Right now, it focuses on three simple areas:

### Features:

- Add tasks with deadlines  
- Set priority (High / Medium / Low)  
- View and delete tasks  
- Save memory notes (like where I kept things)  
- Search memory notes  
- Plan daily activities (Today Plan)  
- Data is saved using files (so it stays even after closing)

---

## How It Works

- Data is stored in `.txt` files  
- When the program starts → it loads data into memory  
- User interacts through the terminal  
- Changes are saved back to the files

## What I’m Thinking Next

Right now, this is a simple system.

But the idea behind MindLink is bigger than just storing tasks.

I’m interested in exploring how this can become more like an assistant:

- Allowing users to quickly capture thoughts (possibly through voice)  
- Converting those thoughts into structured tasks (basic NLP)  
- Helping plan the next day in a simple way (for example, spending 5 minutes before sleep to organize tasks)  
- Making it easier to move from “thinking” to actually “doing”  

The goal is not to build a complex system,  
but to understand how a simple tool can help reduce mental clutter and improve daily execution.
